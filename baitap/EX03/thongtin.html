HTML (HyperText Markup Language)

HTML là ngôn ngữ đánh dấu siêu văn bản, dùng để tạo cấu trúc cơ bản của trang web. Các thẻ HTML giúp xây dựng các phần tử như tiêu đề, đoạn văn bản, hình ảnh, liên kết, bảng, và nhiều thứ khác nữa.

CSS (Cascading Style Sheets)

CSS được dùng để điều chỉnh giao diện của trang web. Nó giúp bạn thay đổi màu sắc, phông chữ, kích thước và bố cục của các phần tử trên trang. CSS giúp làm cho trang web trở nên đẹp mắt và dễ sử dụng hơn.

JavaScript

JavaScript là ngôn ngữ lập trình dùng để tạo tính năng động cho trang web. Nó cho phép người dùng tương tác với các phần tử trên trang, ví dụ như xử lý sự kiện, thay đổi nội dung trang mà không cần tải lại trang.
